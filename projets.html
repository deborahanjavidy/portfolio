Journal d'Humeur

Application de journal d'humeur personnalisé avec authentification et gestion des émotions.

2025 Fini Java MySQL Application Projet personnel

Application de journal d'humeur développée en Java avec base de données MySQL. Cette application permet aux utilisateurs de s'inscrire et se connecter pour créer un journal personnel où ils peuvent enregistrer leur humeur quotidienne avec des émotions personnalisées. L'application inclut une vue calendrier pour visualiser l'évolution des émotions dans le temps, ainsi que des fonctionnalités de gestion de compte (modification email/mot de passe, suppression de compte). Interface intuitive et sécurisée pour un suivi émotionnel personnel et confidentiel.
